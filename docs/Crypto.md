# Crypto Research

SignalForge will include crypto as a major asset class.

## Goal

The crypto market contains too many projects. SignalForge should help filter the market into a smaller number of strong, high-conviction projects for ongoing research.

## Initial Focus

- BTC - core crypto asset
- ETH - smart contract ecosystem leader
- ONDO - real-world asset tokenization
- TAO / Bittensor - decentralized AI watchlist candidate

## Themes

- Bitcoin and digital store of value
- Ethereum and smart contracts
- RWA tokenization
- AI crypto projects
- DePIN
- Stablecoin infrastructure
- Crypto regulation
- Institutional adoption

## Evaluation Criteria

Each crypto project should be evaluated by:

- Real adoption
- Technology quality
- Developer activity
- Tokenomics
- Regulation risk
- Institutional interest
- Liquidity
- Security
- Ecosystem growth
- Long-term use case

## Classification Idea

Projects can be grouped into:

- A Class - highest conviction
- B Class - strong watchlist
- C Class - high risk
- Eliminate - weak or speculative projects
